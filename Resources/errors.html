Диагностика и нейтрализация синтаксических ошибок

По заданию необходимо реализовать нейтрализацию синтаксических ошибок с помощью метода Айронса.

5.1 Метод Айронса

Основная идея метода — если во входной цепочке в процессе разбора встречается символ, который не соответствует ни одному из ожидаемых символов, то его отбрасывают и продолжают разбор.

Алгоритм нейтрализации включает следующие шаги:

  1. Определяются недостроенные кусты дерева разбора;
  2. Формируется множество L — множество остаточных символов недостроенных кустов;
  3. Из входной цепочки удаляется следующий символ до тех пор, пока цепочка не примет вид Tt, такой, что U ⇒ T, где U ∈ L, то есть пока следующий символ T не может быть выведен из какого-либо остаточного символа;
  4. Определяется, какой куст стал причиной появления символа U в множестве L.

Таким образом, определяется, к какому кусту в дереве разбора можно «привязать» оставшуюся часть входной цепочки символов после удаления ошибочного фрагмента.

5.2 Метод Айронса для автоматной грамматики

Рассмотрим дерево разбора с использованием автоматной грамматики:

При возникновении синтаксической ошибки в процессе разбора по автоматной грамматике, в дереве разбора всегда будет только один недостроенный куст.

Так как куст один, он и будет тем самым, к которому можно «привязать» оставшуюся входную цепочку символов. Это позволяет реализовать метод Айронса в виде последовательного удаления символов из входной цепочки до тех пор, пока следующий символ не станет допустимым на текущем этапе разбора.
